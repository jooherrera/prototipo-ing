# TP Ingeniería - Prueba de concepto

### Requisitos:

- Tener instalado NODE.

### Instalación
- Dentro de la ruta raiz ejecutar:
-   `npm i`
-   `npm start`

--- 

La app estará disponible en: http://localhost:3000/


---

### NOTA

Utilizar el botón de `Cerrar sesión` para eliminar los datos guardados.
